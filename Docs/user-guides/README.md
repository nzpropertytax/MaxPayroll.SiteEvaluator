## Quick Overview

### Site Evaluation Wizard

The 8-step wizard guides you through property due diligence:

| Step | Name | Purpose |
|------|------|---------|
| 1 | Address | Enter property location and intended use |
| 2 | Property Match | Select from matching properties |
| 3 | Zoning | Review planning rules and constraints |
| 4 | Hazards | Assess natural hazard risks |
| 5 | Geotech | Review geotechnical data |
| 6 | Infrastructure | Check services availability |
| 7 | Climate | Review wind and rainfall data |
| 8 | Summary | Generate reports |

### Who Uses This?

- Civil Engineers
- Geotechnical Engineers
- Property Developers
- Valuers
- Real Estate Professionals
- Homebuyers
